# Go Win RDC - Integrate PawaPay as the Preferred Payment Gateway

## Project Overview

The Go Win RDC platform is already live and operational.

The current deposit and withdrawal system is manual.

I want to integrate the PawaPay API as the primary payment gateway for all mobile money deposits and withdrawals.

The platform supports two wallet currencies:

* CDF (Congolese Franc)
* USD (United States Dollar)

PawaPay also supports transactions in these currencies, therefore the user's original wallet currency must always be maintained throughout the payment process.

The integration documentation is available here:

https://docs.pawapay.io/v2/docs/welcome

Read the documentation thoroughly and implement the integration according to PawaPay best practices.

---

# Main Objectives

Replace the current manual payment workflow with an automated PawaPay integration while preserving the existing wallet system and administrative approval workflow.

The current betting system, wallet logic, promotions, bonuses, and transaction history must continue functioning.

Only the payment processing mechanism should change.
